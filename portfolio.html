O que você saberá no final do curso

Modelagem no Blender

Após concluir o curso, você saberá construir seus própios modelos 3D no blender para usar em seus jogos.

O curso oferece uma ótima base para fazer seus própios modelos, o que facilita na especialização do blender e melhora seu currículo.

Introdução à Programação

Você também aprenderá os fundamentos da programação utilizando o Scratch, uma linguagem de programação criada pelo MIT em 2007 com o intuito de ensinar a lógica de programação para iniciantes.

Isso lhe dará um bom começo para se aprofundar mais na área da programação de jogos futuramente.

Jogos 3D Com Unity

Uma das principais partes do curso é a parte de desenvolvimento de jogos utilizando Unity, algo vital para um desenvolvedor de jogos.

No curso, o educando aprende a criar jogos 3D nessa plataforma que é uma das principais do mercado de jogos.

Jogos 2D Com Game Maker

Assim como Unity, o Game Maker é um motor de jogo. Mo entanto ele foca mais em desenvolvimento de jogos 2D.

O curso ensina a utilizar e fazer jogos no Game Maker. O que é fundamental para o desenvolvedor de jogos.
